Pour gagner du temps, placez les fenêtres de manière à avoir simultanément sur l'écran le terminal et le navigateur web. Vous pouvez réduire/augmenter la taille des caractères en tapant control - ou +.

Un navigateur web respectant les standards est requis, par exemple du meilleur au moins bon :

  • mozilla, firefox, galeon : Parfait
  • opera : les liens ne peuvent prendre le focus.
  • konqueror : Ne supporte pas l'attribut 'display' du CSS.
  • Internet Explorer : Ne supporte par l'élément ':before' du CSS, il est donc inutilisable pour ce TP.
  • xemacs, lynx, chimera2, dillo, links, links2, amaya : inutilisable car non graphique ou trop ancien.
