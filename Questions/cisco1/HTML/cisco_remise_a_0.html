Effacer la configuration d'un routeur CISCO

Si durant la procédure on vous demande un mot de passe, essayez 'cisco', 'class', 'classe', 'cisco2', '<cisco>'...

Voici la liste de choses à taper (ne faites pas un copier/coller de l'ensemble des lignes). Faites néanmoins attention aux questions que le routeur vous pose. 

enable
erase startup-config
y
reload
                        Tapez return pour confirmer
no
y

Surtout ne pas entrer dans le dialogue pour faire la configuration initiale.
