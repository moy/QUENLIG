Lancement de Quenlig pour les TP CISCO

Voici les opérations qui doivent être faites :

  • Se connecter sous exco sur la machine FAI.
  • Si la version Quenlig a changé : lancer Shell/install_tp_cisco_lif8 (cette commande génère aussi les sessions de TP)
  • Si c'est la première fois que le groupe est en TP alors indiquer le nombre de personnes (machines) dans le fichier :
    Students/cisco[12]/nr_hosts
    • [12] indique le numéro du sujet de TP.
  • Lancer QUENLIG : cd QUENLIG ; ./main.py cisco[12] start
  • Le sujet du TP est dans : QUENLIG/Questions/cisco[12]/HTML/sujet.html
  • Pour être administrateur Quenlig, utilisez le nom exco dans guest.html

ATTENTION :

  • Les outils graphiques (graphviz, inkscape) n'étant pas installé sur FAI le graphique du réseau n'est pas généré. C'est donc moi qui le fait pour vous sur une autre machine.
  • Ne pas changer le nombre d'étudiants du groupe d'une séance à l'autre. Sinon il ne peuvent pas réutiliser leur travail.
  • Le sujet de TP devrait être généré dans Students/cisco[12]/HTML car il dépend de la session. C'est promis, je le corrigerai.

Si cela ne fonctionne pas :

  • L'étudiant n'a pas tapé l'URL complète (il a oublié le numéro de port ou bien guest.html)
  • L'étudiant utilise un navigateur autre que firefox dans lequel le proxy n'est pas configuré.
  • L'étudiant se trompe de routeur.
  • Les câbles routeur/PC ne sont pas branchés sur la bonne paire de machine.
