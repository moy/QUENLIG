Installation de l'IOS sur un routeur CISCO

Cette procédure est nécessaire quand on ne peut plus démarrer l'IOS.

Votre invite de commande doit être rommon...

Choisissez le bon IOS en fonction du modèle de votre routeur

N'oubliez pas de brancher le câble réseau. 

IP_ADDRESS=10.0.0.9
IP_SUBNET_MASK=255.0.0.0
TFTP_FILE=c1700-y-mz.122-11.T11.bin
TFTP_FILE=c1841-ipbase-mz.124-1c.bin
TFTP_SERVER=10.0.0.1
DEFAULT_GATEWAY=10.0.0.1
confreg 0x2142
tftpdnld
boot

Si la carte flash n'est pas utilisable, faites tftpdnld -r
