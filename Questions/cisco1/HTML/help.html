La plupart des problèmes que vous aurez avec les routeurs CISCO est que suivant votre mode les commandes utilisables ne seront pas les même.

Attention, quand on change de mode, on perd les commandes du mode précédent.

N'oubliez pas de lire la documentation CISCO et surtout l'index total du module 2.

Quelques procédures utiles :

  • Effacer les mots de passe : Si vous n'arrivez pas à les trouver.
  • Installer l'IOS : Si vous êtes bloqué sur le ROMMON.
  • Effacer la configuration : Pour enlever le travail des étudiants précédents.
Vous pouvez charger le sujet en PostScript pour le TP 1 Routeur CISCO.
