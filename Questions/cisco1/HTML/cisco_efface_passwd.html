Effacer les mots de passe du routeur CISCO

Pour effacer le mot de passe. Il faut passer en mode ROMMON lors du démarrage puis taper les commandes suivantes : 

confreg 0x2142
reset
no

enable
copy running-config startup-config

reload
no
Remarques :
  • Faire un copié/collé de TOUTE les lignes ne fonctionnera pas.
  • Un ligne vide veux dire que l'on répond Return à la question.
